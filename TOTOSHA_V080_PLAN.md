# TOTOSHA v080 — стабилизация Production

- версия проекта и Release Gate приведены к v080;
- проверка Production ориентирована на новый премиальный интерфейс;
- на главную добавлена интерактивная карта;
- добавлены кнопки маршрута через 2ГИС и Яндекс Карты;
