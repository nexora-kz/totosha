# TOTOSHA Stable Web v037

Дата фиксации: 2026-06-21

## Статус

Контрольная точка стабильного состояния сайта после исправления SEO-навигации и грубых переходов.

## Что закреплено

- Главная страница сохраняет премиальный стиль Тотоша.
- Верхнее меню открывает реальные URL:
  - `/about`
  - `/programs`
  - `/parents`
  - `/cabinet`
  - `/franchise`
  - `/contacts`
- Страница `/life` доступна как фотоархив.
- Мусорный видимый блок `Разделы сайта` удалён.
- Скрытая индексируемая перелинковка сохранена в layout.
- Sitemap и robots остаются частью Release Gate.
- Release Gate проверяет реальные страницы и совпадение live commit.

## Обязательное правило релиза

Нельзя считать релиз готовым только по `Vercel: success`.

Релиз считается готовым только при цепочке:

1. GitHub commit создан.
2. Vercel status = `success`.
3. Live Release Gate вернул `gate: pass`.
4. Live commit совпадает с ожидаемым commit.
5. Все основные страницы открываются.

После PASS можно переходить к финальной SEO-подаче в Google Search Console и Яндекс Вебмастер без изменения дизайна сайта.
