TOTOSHA v21 ASCII Launcher

Use only these launch files:

1) First try:
START_TOTOSHA_ASCII.cmd

It will:
- set npm registry
- install dependencies if node_modules/next is missing
- run the website

Open:
http://localhost:3000

If npm install fails, run:
INSTALL_DEPENDENCIES_ASCII.cmd

Then run:
START_TOTOSHA_ASCII.cmd

Important:
Old START_TOTOSHA.bat files were removed because Windows CMD broke Cyrillic encoding.
