# TOTOSHA SEO scripts

- `monitor.mjs` — технический аудит живого сайта.
- `indexnow.mjs` — уведомление IndexNow после изменения страниц.
- `gsc.mjs` — Search Console Analytics, sitemap и URL Inspection.
- `telegram.mjs` — сводка в Telegram.

Все скрипты используют только встроенные возможности Node.js 20 и не добавляют runtime-зависимости к сайту.
