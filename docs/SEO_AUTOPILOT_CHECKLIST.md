# Checklist подключения TOTOSHA SEO Autopilot

- [ ] Проверить `/icon.svg` после Vercel deployment.
- [ ] Проверить файл IndexNow key на корне домена.
- [ ] Добавить `GOOGLE_SERVICE_ACCOUNT_JSON` в GitHub Secrets.
- [ ] Выдать service account доступ к `https://www.totoshakids.kz/` в Search Console.
- [ ] Проверить `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`.
- [ ] Запустить workflow вручную.
- [ ] Скачать первый artifact и проверить отчёт.
- [ ] Убедиться, что IndexNow вернул 200 или 202.
- [ ] Через 3–7 дней проверить favicon в Яндекс Вебмастере.
- [ ] Не менять URL, robots и sitemap во время стабилизации.
