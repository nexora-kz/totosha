# TOTOSHA SEO Autopilot v065

## Что автоматизировано

1. Ежедневная проверка восьми основных страниц.
2. Контроль HTTP, времени ответа, title, description, canonical, H1 и noindex.
3. Контроль robots.txt, sitemap и favicon.
4. Проверка внутренних ссылок.
5. Отправка изменённых URL в IndexNow после push в `main`.
6. Получение данных Google Search Console при подключении service account.
7. Telegram-отчёт.
8. Хранение отчёта GitHub Actions 30 дней.

## Безопасные ограничения

- система не создаёт фальшивые клики;
- не генерирует отзывы;
- не покупает ссылки;
- не меняет robots.txt и sitemap автоматически;
- не публикует AI-контент без подтверждения;
- IndexNow вызывается после изменения сайта, а не бесконечно по расписанию.

## GitHub Secrets

### Уже используемые проектом

- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`

### Для Google Search Console

- `GOOGLE_SERVICE_ACCOUNT_JSON`

Значение — полный JSON service account в одной GitHub Secret.

Service account нужно добавить как пользователя с правом чтения свойства:

```text
https://www.totoshakids.kz/
```
